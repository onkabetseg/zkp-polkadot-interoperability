Supervisor Demo Commands

1. Open project
cd ~/zkp-polkadot-interoperability
pwd
ls -ltr

2. Show implementation summary
cat results/result10_implementation_results_report/supervisor_summary.txt

3. Show proof artifacts
cd ~/zkp-polkadot-interoperability/zkp-circuits
ls -lh proof_commitment.json public_commitment.json verification_key_commitment.json
cat public_commitment.json

4. Run full Rust Groth16 verification
cd ~/zkp-polkadot-interoperability/rust-groth16-verifier
cargo run

5. Show Result 5 summary
cat ~/zkp-polkadot-interoperability/results/result5_full_rust_groth16_verification/result5_summary.txt

6. Show Substrate verifier adapter
cd ~/zkp-polkadot-interoperability
cat substrate-verifier-pallet/src/verifier/groth16_adapter.rs

7. Show runtime model
cat substrate-verifier-pallet/src/runtime_model/execution_model.rs
cat substrate-verifier-pallet/src/runtime_model/proof_types.rs

8. Show interoperability logic
cat substrate-verifier-pallet/src/interoperability/action.rs

9. Show XCM simulation
cat substrate-verifier-pallet/src/xcm_simulation/message.rs

10. Show full results report
cat results/result10_implementation_results_report/implementation_results_report.md
